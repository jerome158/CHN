Git Clone 使用方法

将镜像前缀添加到原始GitHub地址前面：


原始地址：

git clone https://github.com/user/repo.git

使用镜像：

git clone <镜像前缀>user/repo.git

例如：

git clone https://mirror.ghproxy.com/user/repo.git
